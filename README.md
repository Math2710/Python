# Python — Estudos e Exercícios

Repositório com meus estudos de **Python**, organizado por tópico e seguindo a ordem em que fui aprendendo cada assunto. Cada pasta corresponde a um tema e contém tanto os exemplos das aulas quanto exercícios que resolvi por conta própria para fixar o conteúdo.

Todo o código é escrito à mão, sem bibliotecas externas — o foco aqui é a base da linguagem: sintaxe, tipos, operadores e lógica de programação.

## Tecnologias

- Python 3.14
- Apenas a biblioteca padrão — sem dependências externas

## Como rodar

Cada arquivo é um script independente. Para executar qualquer exercício:

```bash
python for_range/ex1_tabuada.py
```

Trocando o caminho, o mesmo comando serve para qualquer exercício do repositório.

## Convenções

- **Pastas e arquivos** em `snake_case`, minúsculo e sem acentos (PEP 8).
- Arquivos com prefixo `ex1`, `ex2`... são exercícios resolvidos; o arquivo com o nome da pasta é o exemplo da aula.
